# Pattern System Prompt

Este archivo puede usarse para personalizar el comportamiento del motor de mejora de prompts.

Si este archivo existe, su contenido se inyectará como el System Prompt antes de las instrucciones predeterminadas.

## Instrucciones de personalización:

1. Edita este archivo para cambiar el comportamiento base del modelo
2. Usa un estilo claro y directo
3. Especifica el tono, formato y expectativas de salida
4. Guarda los cambios y reinicia la extensión

## Ejemplo:

```
You are an expert prompt engineer specializing in creating clear, actionable prompts for AI systems.
Your improved prompts should be:
- Specific and unambiguous
- Structured with clear sections
- Include explicit input/output specifications
- Use appropriate technical terminology
```
